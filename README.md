# ttu_group_project



## Expected findings

* Core skills such as Python, SQL, and statistics will have the highest frequency across job postings, indicating baseline requirements for data science roles. 
* Job postings requiring advanced skill bundles (e.g., machine learning, cloud platforms, big data tools) will be associated with higher salary ranges. 
* Remote positions will more frequently require specialized and self-directed skills compared to non-remote roles, which will emphasize broader skill sets. 
* Job postings in technology and finance industries will exhibit higher average salaries and more advanced skill requirements than other sectors. 
* Company-level attributes (e.g., size, rating, and revenue) will positively correlate with offered salary ranges in job listings.
